方案 B — 单行标题栏（主题适配）

浅色和暗色模式下的同一布局。所有颜色来自 CSS 令牌，跟随主题切换。

浅色主题
◆ 我的仓库
暗色主题
◆ 我的仓库

布局结构

┌─ 单行标题栏 (40px) ──────────────────────────────────────────┐
│ ◆ 仓库名        + 新建  ⚙  ☰  │  —  □  ✕                   │
│ ← drag-region →  ← 操作区 →   │  ← 窗口控制 →               │
└──────────────────────────────────────────────────────────────┘
  • 整行设置 data-tauri-drag-region，可拖拽移动窗口
  • 窗口控制按钮（最小化/最大化/关闭）调用了 Tauri 窗口 API
  • 所有颜色使用 var(--surface)、var(--text) 等 CSS 令牌，自动适配主题
  • 原 Topbar 组件合并到标题栏中，释放 48px 垂直空间
  • 窗口圆角（Windows 11 风格）通过 Tauri shadow: false + 自绘实现

窗口控制按钮 hover 效果

最大化/最小化 hover 时背景用 --surface-hover；关闭按钮 hover 时红色背景

— hover → 灰色底
✕ hover → #C42B1C 红底 + 白字
