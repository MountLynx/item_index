# i18n — 中英文语言切换

**日期**: 2026-07-13  
**状态**: 设计完成

---

## 1. 概述

使用 `vue-i18n@10` 实现中英文切换，覆盖所有 UI 框架文本和预设数据（预置类别名、字段名）。用户创建的数据不翻译。

## 2. 技术选型

- `vue-i18n@10` — Vue 3 官方国际化库
- 翻译字典：按模块分组的 TypeScript 文件
- 持久化：`locale` 存入 localStorage `index-settings.general.locale`

## 3. 文件结构

```
src/locales/zh-CN.ts   ← 中文翻译字典
src/locales/en.ts      ← 英文翻译字典
src/plugins/i18n.ts    ← createI18n() 初始化
src/main.ts            ← app.use(i18n)
src/stores/settings.ts ← 新增 general section (locale)
```

## 4. 翻译字典

按组件/模块分组，详见 spec 附录。覆盖范围：
- Topbar, Sidebar, CenterList, RightPanel, StatusBar
- SettingsModal（通用 + 主题两个页签全部文本）
- TypeManager, NewItemDialog, EmptyState, Toast
- PropertiesForm, FileTree, IconPicker
- 预设类别名、预设字段名

## 5. 语言切换 UI

SettingsModal → 通用页签，在占位文字上方：

```
  语言    [中文 ▾]
```

`<select>` 下拉，选项为 "中文" / "English"，切换即时生效（watch locale → i18n.global.locale）。

## 6. 持久化

```json
// index-settings
{
  "theme": {...},
  "general": { "locale": "zh-CN" }
}
```

## 7. 组件中使用

```vue
<span>{{ $t('topbar.newItem') }}</span>

<script setup>
import { useI18n } from 'vue-i18n'
const { t } = useI18n()
toast.success(t('typeManager.created'))
</script>
```

## 8. 预设数据翻译

预置类别（id=1 "通用", id=2 "任务"）和预置字段（"名称"、"描述"等）在 typeStore 获取后根据 locale 显示翻译名。后端数据库存中文原文不变。
